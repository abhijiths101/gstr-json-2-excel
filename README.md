# GSTR 1 Utility

## by Shan.tk

Open Source Gstr 1 Json to Excel utility based on Python. Useful for Auditors in Verifying GSTR 1 Return Invoices with Ledgers

### Currently Working Methods

* B2B Invoices
* B2B Credit Notes
* Exports
* B2C Sales
* B2B Amended Invoices

### To be Done
